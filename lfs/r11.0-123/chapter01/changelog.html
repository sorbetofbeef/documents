1.3. Changelog

This is version r11.0-123 of the Linux From Scratch book, dated January 17th, 2022. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via https://www.linuxfromscratch.org/mirrors.html.
